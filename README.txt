User friendly plugin for WooCommerce with single page checkout which facilitates users to shop instantly and to reorder their previously purchased products.
WooCommerce 5.6.x compatible.

== Description ==

Instant Shop for WooCommerce by <a href="https://www.presstigers.com">PressTigers</a> is an easy to use and user friendly plugin that helps users to shop instantly without going to cart and checkout pages.
The plugin also offers features for users to reorder the products which they have ordererd in the past.

= Plugin Features =

*  Easy and Fast one page checkout to place order for a product quickly.
*  Show instant checkout on single product page.
*  Responsive Design.
*  Translation Ready.
*  Allow customers to reorder products again from orders page on My Account.
*  Provide options to select single or multiple products for reordering again.
*  Admin options to select order status for reordering products.

== Installation ==

1. Download plugin.
1. Upload `instant-shop-for-woocommerce.zip` to the `/wp-content/plugins/` directory to your web server.
1. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Which order statuses support Order Again functionality? =
There are two statuses

1.  Processing
2.  Completed

= How checkout will show in single product page? =
When you click Add to Cart button on single product page, it will show the checkout form under product description to purchase the product instantly without going to Cart and Checkout pages.
